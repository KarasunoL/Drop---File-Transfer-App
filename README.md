## About

Drop is a modern file transfer app designed specifically for Android TV.

Built with Jetpack Compose for TV, Drop provides a simple and intuitive way to send and receive files between devices with an interface optimized for TV screens and remote navigation.

## Features

* TV-first file transfer experience
* Send files from Android TV
* Receive files on Android TV
* Remote-friendly navigation
* Simple device discovery
* Clean transfer interface
* Transfer progress tracking
* TV-optimized file selection
* Clean and modern interface

## Upcoming Updates

* Android phone support
* Improved device discovery
* Transfer history
* Improved file selection
* Faster and more reliable transfers
* Background transfer improvements
* More customization options

## Built With

* Kotlin
* Jetpack Compose for TV
* Material 3
* Android Networking APIs

## Platform

Drop is currently designed for **Android TV**.

Android phone support will come later.
